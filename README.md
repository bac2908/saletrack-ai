# SaleTrackAI

SaleTrackAI is a sales tracking workspace with a React frontend and an Express/Prisma backend.

## Stack

- Frontend: React, TypeScript, Vite, Tailwind CSS
- Backend: Node.js, Express, TypeScript
- Database: SQLite
- ORM: Prisma

## Structure

- `frontend/` contains the Vite React application.
- `backend/` contains the Express API and Prisma data layer.
- `docs/` contains project documentation.
- `screenshots/` stores application screenshots.

## Getting Started

```bash
npm install
npm run dev:frontend
npm run dev:backend
```
